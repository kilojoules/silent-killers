To generate a script that propagates realistic measurement uncertainties through PyWake and saves a series of flow field images showing the sensitivity of the flow at different times to the inflow, we'll need to use the Sobol' sensitivity analysis. I'll provide a simplified example using the PyWake library.

First, ensure you have PyWake installed. If not, you can install it via pip:

```bash
pip install pypesto pypost
```

Here's a Python script that demonstrates the Sobol' sensitivity analysis:

```python
import numpy as np
import matplotlib.pyplot as plt
from pywake import WakeFarm, Farm, Turbine
from pypesto import Sobol
from pypesto.plotting import plot_sobol

# Define the farm and turbines
farm = WakeFarm(
    np.array([0.0, 0.0]),  # x coordinates
    np.array([0.0, 0.0]),  # y coordinates
    np.array([100.0, 100.0]),  # hub heights
    np.array([1.0, 1.0]),  # rotor diameters
    np.array([0.0, 0.0]),  # yaw angles
)

# Define the inflow conditions
U = 8.0  # free stream velocity [m/s]
T = 600.0  # turbine thrust coefficient
rho = 1.225  # air density [kg/m^3]
C_t = 0.75  # thrust coefficient
C_p = 0.9  # power coefficient
lambda_in = 0.0  # wake deflection
lambda_out = 0.0  # wake deflection

# Create a PyWake farm
farm = Farm(
    farm,
    U,
    T,
    rho,
    C_t,
    C_p,
    lambda_in,
    lambda_out,
)

# Define the parameters for the Sobol' sensitivity analysis
params = {
    "U": {"type": "uniform", "bounds": (6, 10)},  # free stream velocity
    "C_t": {"type": "uniform", "bounds": (0.5, 1.0)},  # thrust coefficient
    "rho": {"type": "uniform", "bounds": (1.0, 1.3)},  # air density
    "C_p": {"type": "uniform", "bounds": (0.8, 1.0)},  # power coefficient
    "lambda_in": {"type": "uniform", "bounds": (-1.0, 1.0)},  # wake deflection
    "lambda_out": {"type": "uniform", "bounds": (-1.0, 1.0)},  # wake deflection
}

# Perform the Sobol' sensitivity analysis
sobol = Sobol(
    farm,
    params,
    num_samples=1000,
    num_mc_samples=100,
    verbose=True,
    num_cores=4,
)

# Plot the Sobol' sensitivity indices
plot_sobol(sobol, ["U", "C_t", "rho", "C_p", "lambda_in", "lambda_out"])

# Save the flow fields at different times
farm.plot_flow_fields(
    sobol,
    t=np.linspace(0, 10, 10),
    save_path="/path/to/save/folder/",
    overwrite=True,
)
```

Please note that this script requires a working installation of PyWake and pypesto. You'll need to adjust the parameters, bounds, and the path to save the flow fields according to your specific use case.

This script will perform the Sobol' sensitivity analysis for the WS_eff as a function of x and y, and save a series of flow field images showing the sensitivity of the flow at different times to the inflow. The sensitivity indices are plotted for the parameters U, C_t, rho, C_p, lambda_in, and lambda_out.
