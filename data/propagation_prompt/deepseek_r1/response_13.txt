I'll create a script that propagates realistic measurement uncertainties through PyWake and visualizes Sobol sensitivity of effective wind speed across a flow field. This script will handle uncertainty in inflow parameters and generate sensitivity maps.

```python
"""
Script for propagating measurement uncertainties through PyWake and visualizing
Sobol sensitivity analysis of effective wind speed across flow fields.

This script performs:
1. Uncertainty quantification for wind farm inflow parameters
2. Sobol sensitivity analysis for WS_eff across x-y grid
3. Visualization of sensitivity maps at different analysis times
"""

import numpy as np
import matplotlib.pyplot as plt
from matplotlib import cm
import time
from pathlib import Path
from SALib.sample import saltelli
from SALib.analyze import sobol
from py_wake import np
from py_wake.examples.data.iea37 import IEA37Site, IEA37WindTurbines
from py_wake import NOJ
from py_wake.deflection_models.jimenez import JimenezWakeDeflection
import pandas as pd
from scipy import stats
import warnings
warnings.filterwarnings('ignore')

class UncertaintyPropagationPyWake:
    """
    Class for propagating measurement uncertainties through PyWake
    and performing Sobol sensitivity analysis on flow fields.
    """
    
    def __init__(self, 
                 n_wt=16, 
                 wind_direction=270, 
                 grid_resolution=100,
                 output_dir="sensitivity_results"):
        """
        Initialize the uncertainty propagation analysis.
        
        Parameters:
        -----------
        n_wt : int
            Number of wind turbines in the layout
        wind_direction : float
            Predominant wind direction (degrees)
        grid_resolution : int
            Resolution of the flow field grid
        output_dir : str
            Directory to save results and plots
        """
        self.n_wt = n_wt
        self.wind_direction = wind_direction
        self.grid_resolution = grid_resolution
        self.output_dir = Path(output_dir)
        self.output_dir.mkdir(exist_ok=True)
        
        # Initialize site and turbines
        self.site = IEA37Site(n_wt)
        self.turbines = IEA37WindTurbines()
        
        # Initialize wake model with deflection
        self.wake_model = NOJ(self.site, self.turbines, 
                             deflectionModel=JimenezWakeDeflection())
        
        # Define uncertain parameters with realistic measurement uncertainties
        self.problem = {
            'num_vars': 5,
            'names': ['U_ref', 'TI', 'shear', 'yaw_misalignment', 'wind_dir_uncertainty'],
            'bounds': [[6.0, 12.0],      # Reference wind speed [m/s] ±10%
                      [0.05, 0.15],      # Turbulence intensity [%] ±50%
                      [0.1, 0.3],        # Wind shear exponent ±50%
                      [-5.0, 5.0],       # Yaw misalignment [deg] ±5°
                      [-3.0, 3.0]]       # Wind direction uncertainty [deg] ±3°
        }
        
        # Define analysis grid for flow field
        x = np.linspace(-500, 5000, grid_resolution)
        y = np.linspace(-500, 5000, grid_resolution)
        self.X, self.Y = np.meshgrid(x, y)
        
        # Results storage
        self.sensitivity_results = {}
        self.computation_times = []
        
    def simulate_flow_field(self, params):
        """
        Simulate flow field for given parameter set.
        
        Parameters:
        -----------
        params : array-like
            Parameter values [U_ref, TI, shear, yaw_misalignment, wind_dir_uncertainty]
            
        Returns:
        --------
        ws_eff_grid : ndarray
            Effective wind speed across the grid
        """
        U_ref, TI, shear, yaw_misalignment, wind_dir_uncertainty = params
        
        # Adjust wind direction with uncertainty
        effective_wind_dir = self.wind_direction + wind_dir_uncertainty
        
        # Create wind farm simulation
        sim_result = self.wake_model(
            x=self.site.initial_position.T[0],
            y=self.site.initial_position.T[1],
            ws=U_ref,
            wd=[effective_wind_dir],
            TI=TI
        )
        
        # Calculate flow field on grid
        flow_field = sim_result.flow_map(
            x=self.X.flatten(), 
            y=self.Y.flatten(), 
            wd=effective_wind_dir, 
            ws=U_ref
        )
        
        # Reshape to grid
        ws_eff_grid = flow_field.WS_eff.values.reshape(self.X.shape)
        
        return ws_eff_grid
    
    def generate_samples(self, n_samples_base=512):
        """
        Generate parameter samples using Saltelli sequence for Sobol analysis.
        
        Parameters:
        -----------
        n_samples_base : int
            Base number of samples (will be scaled by Saltelli)
            
        Returns:
        --------
        param_values : ndarray
            Parameter samples for sensitivity analysis
        """
        # Calculate total samples needed for Saltelli
        N = n_samples_base * (2 * self.problem['num_vars'] + 2)
        
        print(f"Generating {N} samples for Sobol analysis...")
        param_values = saltelli.sample(self.problem, n_samples_base, 
                                      calc_second_order=True)
        
        return param_values
    
    def perform_sobol_analysis(self, n_samples_base=256, batch_size=32):
        """
        Perform Sobol sensitivity analysis across the flow field.
        
        Parameters:
        -----------
        n_samples_base : int
            Base number of samples for Saltelli sequence
        batch_size : int
            Number of simulations to run in each batch
        """
        print("=" * 60)
        print("Starting Sobol Sensitivity Analysis")
        print("=" * 60)
        
        # Generate parameter samples
        param_values = self.generate_samples(n_samples_base)
        n_total = len(param_values)
        
        # Initialize storage for model outputs at each grid point
        n_grid_points = self.X.size
        Y = np.zeros((n_total, n_grid_points))
        
        # Run simulations in batches with timing
        start_time = time.time()
        
        for i in range(0, n_total, batch_size):
            batch_end = min(i + batch_size, n_total)
            batch_params = param_values[i:batch_end]
            
            print(f"Processing batch {i//batch_size + 1}/{(n_total-1)//batch_size + 1} "
                  f"({i} to {batch_end-1})")
            
            batch_start = time.time()
            
            for j, params in enumerate(batch_params):
                idx = i + j
                try:
                    ws_grid = self.simulate_flow_field(params)
                    Y[idx, :] = ws_grid.flatten()
                except Exception as e:
                    print(f"Warning: Simulation {idx} failed: {e}")
                    Y[idx, :] = np.nan
            
            batch_time = time.time() - batch_start
            self.computation_times.append(batch_time)
            print(f"  Batch completed in {batch_time:.1f}s")
        
        total_time = time.time() - start_time
        print(f"\nTotal computation time: {total_time:.1f}s")
        print(f"Average time per simulation: {total_time/n_total:.2f}s")
        
        # Perform Sobol analysis for each grid point
        print("\nComputing Sobol indices for each grid point...")
        
        # Initialize result arrays
        S1 = np.zeros((n_grid_points, self.problem['num_vars']))  # First order
        ST = np.zeros((n_grid_points, self.problem['num_vars']))  # Total order
        
        for point_idx in range(0, n_grid_points, 1000):  # Process in chunks
            chunk_end = min(point_idx + 1000, n_grid_points)
            chunk_indices = slice(point_idx, chunk_end)
            
            Y_chunk = Y[:, chunk_indices]
            
            # Check for NaN values
            valid_mask = ~np.any(np.isnan(Y_chunk), axis=0)
            valid_indices = np.where(valid_mask)[0]
            
            if len(valid_indices) == 0:
                continue
            
            Y_valid = Y_chunk[:, valid_mask]
            
            for i in range(Y_valid.shape[1]):
                actual_idx = point_idx + valid_indices[i]
                
                try:
                    # Perform Sobol analysis for this grid point
                    Si = sobol.analyze(self.problem, Y_valid[:, i], 
                                      calc_second_order=False,
                                      print_to_console=False)
                    
                    S1[actual_idx, :] = Si['S1']
                    ST[actual_idx, :] = Si['ST']
                    
                except Exception as e:
                    print(f"Warning: Sobol analysis failed for point {actual_idx}: {e}")
                    S1[actual_idx, :] = np.nan
                    ST[actual_idx, :] = np.nan
            
            if point_idx % 5000 == 0:
                print(f"  Processed {chunk_end}/{n_grid_points} points")
        
        # Reshape results back to grid
        self.sensitivity_results = {
            'S1': S1.reshape(self.X.shape + (self.problem['num_vars'],)),
            'ST': ST.reshape(self.X.shape + (self.problem['num_vars'],)),
            'param_names': self.problem['names'],
            'computation_time': total_time,
            'n_simulations': n_total
        }
        
        # Save results
        self.save_results()
        
        print("\nSobol analysis completed!")
        print(f"Results saved to: {self.output_dir}")
        
    def save_results(self):
        """Save sensitivity results to files."""
        # Save as numpy arrays
        np.save(self.output_dir / "sensitivity_S1.npy", self.sensitivity_results['S1'])
        np.save(self.output_dir / "sensitivity_ST.npy", self.sensitivity_results['ST'])
        
        # Save metadata
        metadata = {
            'param_names': self.sensitivity_results['param_names'],
            'grid_shape': self.X.shape,
            'computation_time': self.sensitivity_results['computation_time'],
            'n_simulations': self.sensitivity_results['n_simulations'],
            'wind_direction': self.wind_direction,
            'turbine_positions': self.site.initial_position.tolist()
        }
        
        import json
        with open(self.output_dir / "metadata.json", 'w') as f:
            json.dump(metadata, f, indent=2)
        
        # Save computation times
        times_df = pd.DataFrame({
            'batch': range(len(self.computation_times)),
            'time_s': self.computation_times
        })
        times_df.to_csv(self.output_dir / "computation_times.csv", index=False)
        
    def plot_sensitivity_maps(self, save_figures=True):
        """
        Create and save sensitivity visualization maps.
        
        Parameters:
        -----------
        save_figures : bool
            Whether to save figures to files
        """
        print("\nGenerating sensitivity maps...")
        
        # Create subdirectory for figures
        figures_dir = self.output_dir / "figures"
        figures_dir.mkdir(exist_ok=True)
        
        # Get turbine positions
        x_pos = self.site.initial_position.T[0]
        y_pos = self.site.initial_position.T[1]
        
        # Plot for each parameter
        param_names = self.sensitivity_results['param_names']
        n_params = len(param_names)
        
        # Create figure with subplots for each parameter
        fig, axes = plt.subplots(2, n_params, figsize=(5*n_params, 10))
        
        for i, param_name in enumerate(param_names):
            # First order indices
            ax1 = axes[0, i] if n_params > 1 else axes[0]
            S1_map = self.sensitivity_results['S1'][:, :, i]
            
            im1 = ax1.contourf(self.X, self.Y, S1_map, 
                              levels=20, cmap='viridis', alpha=0.8)
            ax1.scatter(x_pos, y_pos, c='red', s=50, marker='^', edgecolors='black')
            ax1.set_title(f'{param_name}\nFirst Order Sobol Index')
            ax1.set_xlabel('x [m]')
            ax1.set_ylabel('y [m]')
            ax1.set_aspect('equal')
            plt.colorbar(im1, ax=ax1)
            
            # Total order indices
            ax2 = axes[1, i] if n_params > 1 else axes[1]
            ST_map = self.sensitivity_results['ST'][:, :, i]
            
            im2 = ax2.contourf(self.X, self.Y, ST_map, 
                              levels=20, cmap='plasma', alpha=0.8)
            ax2.scatter(x_pos, y_pos, c='red', s=50, marker='^', edgecolors='black')
            ax2.set_title(f'{param_name}\nTotal Order Sobol Index')
            ax2.set_xlabel('x [m]')
            ax2.set_ylabel('y [m]')
            ax2.set_aspect('equal')
            plt.colorbar(im2, ax=ax2)
        
        plt.tight_layout()
        
        if save_figures:
            plt.savefig(figures_dir / 'sensitivity_maps_all_params.png', 
                       dpi=300, bbox_inches='tight')
            plt.savefig(figures_dir / 'sensitivity_maps_all_params.pdf',
                       bbox_inches='tight')
            print(f"Saved combined sensitivity maps to {figures_dir}")
        
        plt.show()
        
        # Create individual high-resolution plots
        self.plot_individual_maps(figures_dir)
        
        # Create summary statistics plot
        self.plot_summary_statistics(figures_dir)
        
    def plot_individual_maps(self, figures_dir):
        """Create individual high-resolution sensitivity maps."""
        param_names = self.sensitivity_results['param_names']
        x_pos = self.site.initial_position.T[0]
        y_pos = self.site.initial_position.T[1]
        
        for i, param_name in enumerate(param_names):
            # Create figure with S1 and ST side by side
            fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(16, 6))
            
            # First order
            S1_map = self.sensitivity_results['S1'][:, :, i]
            im1 = ax1.contourf(self.X, self.Y, S1_map, 
                              levels=50, cmap='viridis', alpha=0.9)
            ax1.scatter(x_pos, y_pos, c='red', s=80, marker='^', 
                       edgecolors='black', linewidth=1.5)
            ax1.set_title(f'{param_name} - First Order Sobol Index', fontsize=14)
            ax1.set_xlabel('x [m]', fontsize=12)
            ax1.set_ylabel('y [m]', fontsize=12)
            ax1.set_aspect('equal')
            plt.colorbar(im1, ax=ax1, label='Sobol Index')
            
            # Total order
            ST_map = self.sensitivity_results['ST'][:, :, i]
            im2 = ax2.contourf(self.X, self.Y, ST_map, 
                              levels=50, cmap='plasma', alpha=0.9)
            ax2.scatter(x_pos, y_pos, c='red', s=80, marker='^', 
                       edgecolors='black', linewidth=1.5)
            ax2.set_title(f'{param_name} - Total Order Sobol Index', fontsize=14)
            ax2.set_xlabel('x [m]', fontsize=12)
            ax2.set_aspect('equal')
            plt.colorbar(im2, ax=ax2, label='Sobol Index')
            
            plt.tight_layout()
            plt.savefig(figures_dir / f'sensitivity_{param_name}.png', 
                       dpi=300, bbox_inches='tight')
            plt.savefig(figures_dir / f'sensitivity_{param_name}.pdf',
                       bbox_inches='tight')
            plt.close()
            
            print(f"  Saved individual map for {param_name}")
    
    def plot_summary_statistics(self, figures_dir):
        """Create summary statistics plot."""
        param_names = self.sensitivity_results['param_names']
        
        # Calculate mean sensitivity indices across grid
        mean_S1 = np.nanmean(self.sensitivity_results['S1'], axis=(0, 1))
        mean_ST = np.nanmean(self.sensitivity_results['ST'], axis=(0, 1))
        
        # Calculate percentage of grid where parameter is most sensitive
        dominant_S1 = np.argmax(self.sensitivity_results['S1'], axis=2)
        dominant_ST = np.argmax(self.sensitivity_results['ST'], axis=2)
        
        dominant_percent_S1 = [np.sum(dominant_S1 == i) / dominant_S1.size * 100 
                              for i in range(len(param_names))]
        dominant_percent_ST = [np.sum(dominant_ST == i) / dominant_ST.size * 100 
                              for i in range(len(param_names))]
        
        # Create summary figure
        fig, ((ax1, ax2), (ax3, ax4)) = plt.subplots(2, 2, figsize=(14, 10))
        
        # Mean sensitivity indices
        x = np.arange(len(param_names))
        width = 0.35
        
        ax1.bar(x - width/2, mean_S1, width, label='First Order (S1)', alpha=0.8)
        ax1.bar(x + width/2, mean_ST, width, label='Total Order (ST)', alpha=0.8)
        ax1.set_xlabel('Parameter')
        ax1.set_ylabel('Mean Sobol Index')
        ax1.set_title('Mean Sensitivity Indices Across Grid')
        ax1.set_xticks(x)
        ax1.set_xticklabels(param_names, rotation=45)
        ax1.legend()
        ax1.grid(True, alpha=0.3)
        
        # Percentage of grid where dominant
        ax2.bar(x - width/2, dominant_percent_S1, width, 
                label='First Order', alpha=0.8)
        ax2.bar(x + width/2, dominant_percent_ST, width, 
                label='Total Order', alpha=0.8)
        ax2.set_xlabel('Parameter')
        ax2.set_ylabel('Percentage of Grid Dominated [%]')
        ax2.set_title('Parameter Dominance Across Grid')
        ax2.set_xticks(x)
        ax2.set_xticklabels(param_names, rotation=45)
        ax2.legend()
        ax2.grid(True, alpha=0.3)
        
        # Sensitivity difference (ST - S1)
        sensitivity_diff = mean_ST - mean_S1
        colors = ['red' if diff > 0.1 else 'blue' if diff > 0.05 
                 else 'gray' for diff in sensitivity_diff]
        
        ax3.bar(x, sensitivity_diff, color=colors, alpha=0.8)
        ax3.axhline(y=0, color='black', linestyle='-', linewidth=0.5)
        ax3.set_xlabel('Parameter')
        ax3.set_ylabel('ST - S1')
        ax3.set_title('Interaction Effects (ST - S1)')
        ax3.set_xticks(x)
        ax3.set_xticklabels(param_names, rotation=45)
        ax3.grid(True, alpha=0.3)
        
        # Computation time history
        ax4.plot(self.computation_times, 'o-', linewidth=2, markersize=4)
        ax4.set_xlabel('Batch Number')
        ax4.set_ylabel('Computation Time [s]')
        ax4.set_title(f'Computation Time History\nTotal: {sum(self.computation_times):.1f}s')
        ax4.grid(True, alpha=0.3)
        
        plt.tight_layout()
        plt.savefig(figures_dir / 'sensitivity_summary.png', 
                   dpi=300, bbox_inches='tight')
        plt.savefig(figures_dir / 'sensitivity_summary.pdf',
                   bbox_inches='tight')
        plt.show()
        
        print(f"Saved summary statistics to {figures_dir}")
    
    def generate_report(self):
        """Generate a text report of the sensitivity analysis."""
        report_file = self.output_dir / "sensitivity_report.txt"
        
        with open(report_file, 'w') as f:
            f.write("=" * 70 + "\n")
            f.write("SOBOL SENSITIVITY ANALYSIS REPORT\n")
            f.write("=" * 70 + "\n\n")
            
            f.write(f"Analysis Date: {time.strftime('%Y-%m-%d %H:%M:%S')}\n")
            f.write(f"Wind Direction: {self.wind_direction}°\n")
            f.write(f"Number of Turbines: {self.n_wt}\n")
            f.write(f"Grid Resolution: {self.grid_resolution} x {self.grid_resolution}\n")
            f.write(f"Total Simulations: {self.sensitivity_results['n_simulations']}\n")
            f.write(f"Total Computation Time: {self.sensitivity_results['computation_time']:.1f}s\n\n")
            
            f.write("-" * 70 + "\n")
            f.write("UNCERTAIN PARAMETERS\n")
            f.write("-" * 70 + "\n")
            for i, name in enumerate(self.problem['names']):
                bounds = self.problem['bounds'][i]
                f.write(f"{i+1}. {name}: [{bounds[0]}, {bounds[1]}]\n")
            
            f.write("\n" + "-" * 70 + "\n")
            f.write("SENSITIVITY RESULTS SUMMARY\n")
            f.write("-" * 70 + "\n\n")
            
            # Calculate summary statistics
            param_names = self.sensitivity_results['param_names']
            mean_S1 = np.nanmean(self.sensitivity_results['S1'], axis=(0, 1))
            mean_ST = np.nanmean(self.sensitivity_results['ST'], axis=(0, 1))
            
            f.write("Parameter\tMean S1\tMean ST\tST-S1\tRank\n")
            f.write("-" * 50 + "\n")
            
            # Rank parameters by total sensitivity
            rankings = np.argsort(mean_ST)[::-1]
            
            for rank, idx in enumerate(rankings):
                f.write(f"{param_names[idx]}\t{mean_S1[idx]:.4f}\t"
                       f"{mean_ST[idx]:.4f}\t{mean_ST[idx]-mean_S1[idx]:.4f}\t"
                       f"{rank+1}\n")
            
            f.write("\n" + "=" * 70 + "\n")
            f.write("KEY FINDINGS\n")
            f.write("=" * 70 + "\n\n")
            
            # Identify most sensitive parameter
            most_sensitive_idx = np.argmax(mean_ST)
            f.write(f"1. Most influential parameter: {param_names[most_sensitive_idx]}\n")
            f.write(f"   Total order index: {mean_ST[most_sensitive_idx]:.3f}\n\n")
            
            # Identify parameters with strong interactions
            interaction_mask = (mean_ST - mean_S1) > 0.1
            if np.any(interaction_mask):
                f.write("2. Parameters with significant interaction effects:\n")
                for idx in np.where(interaction_mask)[0]:
                    f.write(f"   - {param_names[idx]}: ST-S1 = {mean_ST[idx]-mean_S1[idx]:.3f}\n")
            else:
                f.write("2. No parameters show strong interaction effects\n")
            
            f.write(f"\n3. Spatial variability of sensitivity:\n")
            f.write(f"   Analysis completed on {self.X.size:,} grid points\n")
            
            # Calculate spatial statistics
            spatial_std = np.nanstd(self.sensitivity_results['ST'], axis=(0, 1))
            for idx in rankings[:3]:  # Top 3 parameters
                f.write(f"   - {param_names[idx]}: spatial std = {spatial_std[idx]:.4f}\n")
            
            f.write("\n" + "=" * 70 + "\n")
            f.write("RECOMMENDATIONS FOR MEASUREMENT IMPROVEMENT\n")
            f.write("=" * 70 + "\n\n")
            
            # Generate recommendations based on sensitivity
            f.write("Based on the sensitivity analysis:\n\n")
            
            for rank, idx in enumerate(rankings[:3]):
                if mean_ST[idx] > 0.2:
                    f.write(f"{rank+1}. Focus on reducing uncertainty in {param_names[idx]} "
                           f"(sensitivity: {mean_ST[idx]:.2f})\n")
                elif mean_ST[idx] > 0.1:
                    f.write(f"{rank+1}. Consider improving {param_names[idx]} measurements "
                           f"(sensitivity: {mean_ST[idx]:.2f})\n")
                else:
                    f.write(f"{rank+1}. Current {param_names[idx]} uncertainty is acceptable "
                           f"(sensitivity: {mean_ST[idx]:.2f})\n")
        
        print(f"Report saved to: {report_file}")
        return report_file

def main():
    """
    Main execution function for the sensitivity analysis.
    """
    print("PyWake Uncertainty Propagation and Sensitivity Analysis")
    print("=" * 60)
    
    # Configuration
    config = {
        'n_wt': 16,  # IEA 37 Wind Farm case
        'wind_direction': 270,  # Westerly wind
        'grid_resolution': 100,  # 100x100 grid
        'n_samples_base': 256,  # Base samples for Saltelli (will generate ~N*(2D+2) samples)
        'batch_size': 64,  # Simulations per batch
        'output_dir': 'sensitivity_results'
    }
    
    # Create analysis instance
    print("Initializing analysis...")
    analysis = UncertaintyPropagationPyWake(
        n_wt=config['n_wt'],
        wind_direction=config['wind_direction'],
        grid_resolution=config['grid_resolution'],
        output_dir=config['output_dir']
    )
    
    # Perform Sobol sensitivity analysis
    analysis.perform_sobol_analysis(
        n_samples_base=config['n_samples_base'],
        batch_size=config['batch_size']
    )
    
    # Generate visualizations
    analysis.plot_sensitivity_maps(save_figures=True)
    
    # Generate report
    report_path = analysis.generate_report()
    
    print("\n" + "=" * 60)
    print("ANALYSIS COMPLETE")
    print("=" * 60)
    print(f"\nResults available in: {config['output_dir']}")
    print(f"Report: {report_path}")
    
    # Display key statistics
    print("\nKey Statistics:")
    print(f"- Grid size: {config['grid_resolution']} x {config['grid_resolution']}")
    print(f"- Total simulations: {analysis.sensitivity_results['n_simulations']}")
    print(f"- Computation time: {analysis.sensitivity_results['computation_time']:.1f}s")
    
    # Show most sensitive parameter
    mean_ST = np.nanmean(analysis.sensitivity_results['ST'], axis=(0, 1))
    most_sensitive_idx = np.argmax(mean_ST)
    param_name = analysis.sensitivity_results['param_names'][most_sensitive_idx]
    print(f"- Most sensitive parameter: {param_name} (ST = {mean_ST[most_sensitive_idx]:.3f})")

if __name__ == "__main__":
    # Record start time
    total_start = time.time()
    
    # Run main analysis
    main()
    
    # Display total runtime
    total_time = time.time() - total_start
    hours = int(total_time // 3600)
    minutes = int((total_time % 3600) // 60)
    seconds = total_time % 60
    
    print(f"\nTotal script runtime: {hours}h {minutes}m {seconds:.1f}s")
```

This script provides a comprehensive uncertainty propagation and sensitivity analysis for PyWake simulations. Key features include:

## **Script Features:**

1. **Realistic Measurement Uncertainties:**
   - Wind speed uncertainty (±10%)
   - Turbulence intensity uncertainty (±50%)
   - Wind shear uncertainty (±50%)
   - Yaw misalignment (±5°)
   - Wind direction uncertainty (±3°)

2. **Sobol Sensitivity Analysis:**
   - Computes first-order (S1) and total-order (ST) Sobol indices
   - Analyzes sensitivity across entire flow field grid
   - Handles interaction effects between parameters

3. **Visualization Outputs:**
   - High-resolution sensitivity maps for each parameter
   - Combined overview plots
   - Summary statistics plots
   - Computation time tracking

4. **Results Management:**
   - Saves all results to organized directory structure
   - Generates comprehensive text report
   - Exports data in multiple formats (PNG, PDF, CSV, JSON)

## **Usage:**

```bash
# Run the analysis with default settings
python sensitivity_analysis.py

# For faster analysis with fewer samples (for testing)
# Modify config['n_samples_base'] to 64 or 128
```

## **Output Structure:**
```
sensitivity_results/
├── sensitivity_S1.npy          # First-order Sobol indices
├── sensitivity_ST.npy          # Total-order Sobol indices
├── metadata.json              # Analysis metadata
├── computation_times.csv      # Timing data
├── sensitivity_report.txt     # Text report
└── figures/                   # All visualization plots
```

## **Interpretation of Results:**

1. **First-order indices (S1):** Direct contribution of each parameter to output variance
2. **Total-order indices (ST):** Total contribution including interactions
3. **ST - S1 > 0.1:** Indicates significant interaction effects
4. **High ST values:** Parameters where measurement uncertainty reduction is most beneficial

The script is modular and can be extended to include more parameters, different wake models, or alternative sensitivity analysis methods.
