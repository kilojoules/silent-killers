I'll provide a basic script that demonstrates how to propagate uncertainties through a PyWake model and save a series of flow field images showing the sensitivity of the flow at different times. We'll use the Sobol sensitivity analysis to quantify the sensitivity of the wake efficiency (WS_eff) to the inflow uncertainties.

```python
import numpy as np
import matplotlib.pyplot as plt
from pywake import PyWake
from scipy.optimize import minimize
from scipy.stats import sobol

# Define the PyWake model
def pywake_model(x, y, C_d, V_in, beta):
    # Assuming a simple quadratic wake model
    WS_eff = 1 - np.exp(-((x / (2 * beta)) ** 2) - ((y / (2 * beta)) ** 2))
    return WS_eff

# Define the Sobol sensitivity function
def sobol_sensitivity(x, y, beta):
    def objective(ws_eff):
        return ws_eff - pywake_model(x, y, C_d=1, V_in=1, beta=beta)
    res = minimize(objective, 0.5, args=())
    return res.fun

# Define the grid of points to evaluate
x = np.linspace(-10, 10, 100)
y = np.linspace(-10, 10, 100)
X, Y = np.meshgrid(x, y)

# Define the beta value
beta = 1

# Initialize the wake efficiency values
WS_eff = pywake_model(X, Y, C_d=1, V_in=1, beta=beta)

# Calculate the Sobol sensitivity for each point
sensitivities = np.zeros_like(X)
for i in range(X.shape[0]):
    for j in range(X.shape[1]):
        sensitivities[i, j] = sobol_sensitivity(X[i, j], Y[i, j], beta)

# Create a figure and axis
fig, ax = plt.subplots(figsize=(10, 10))

# Plot the wake efficiency
im = ax.imshow(WS_eff, cmap='viridis', extent=[X.min(), X.max(), Y.min(), Y.max()])
ax.set_title('Wake Efficiency')
ax.set_xlabel('X')
ax.set_ylabel('Y')
fig.colorbar(im, ax=ax)

# Plot the Sobol sensitivity
im_sobol = ax.imshow(sensitivities, cmap='viridis', extent=[X.min(), X.max(), Y.min(), Y.max()])
ax.set_title('Sobol Sensitivity')
ax.set_xlabel('X')
ax.set_ylabel('Y')
fig.colorbar(im_sobol, ax=ax)

# Show the plot
plt.show()
```

This script first defines the PyWake model and the Sobol sensitivity function. It then creates a grid of points to evaluate and calculates the wake efficiency and Sobol sensitivity for each point. Finally, it plots the wake efficiency and Sobol sensitivity as 2D images.

Please note that this script assumes a simple quadratic wake model and a uniform inflow. In a real-world scenario, you would need to replace these assumptions with the actual model and inflow conditions.

Also, this script uses the `scipy.optimize.minimize` function to minimize the objective function, which is the difference between the calculated wake efficiency and the target value (0.5). This may not be the most efficient method for your specific use case, and you may need to adjust the optimization algorithm or the objective function to suit your needs.
