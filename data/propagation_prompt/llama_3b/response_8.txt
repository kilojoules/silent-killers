To generate a script that propagates realistic measurement uncertainties through pywake and saves a series of flow field images showing the sensitivity of the flow at different times to the inflow, we can use the following Python script. This script uses the Sobol sensitivity analysis to estimate the global sensitivity of the wave propagation and saves the sensitivity of WS_eff as a function of x and y.

```python
import numpy as np
import pywake as pw
import matplotlib.pyplot as plt

# Define the parameters
N_x, N_y = 100, 100  # number of grid points in x and y directions
Lx, Ly = 10, 10  # length of the domain in x and y directions
T = 1.0  # time duration
Nt = 100  # number of time steps
nx = N_x // Nt  # number of grid points in x direction per time step
ny = N_y // Nt  # number of grid points in y direction per time step
alpha = 0.5  # wave speed
beta = 0.1  # wave amplitude
sigma = 0.1  # initial perturbation amplitude
x = np.linspace(0, Lx, N_x)
y = np.linspace(0, Ly, N_y)
X, Y = np.meshgrid(x, y)
dx, dy = np.diff(x), np.diff(y)

# Initialize the flow field
u = np.zeros((N_y, N_x, Nt))
v = np.zeros((N_y, N_x, Nt))
w = np.zeros((N_y, N_x, Nt))

# Initialize the wave
u[0, :, 0] = beta * np.sin(alpha * X) * np.sin(alpha * Y)
v[0, :, 0] = beta * np.cos(alpha * X) * np.sin(alpha * Y)
w[0, :, 0] = 0

# Initialize the perturbations
eps = np.random.normal(0, sigma, (N_y, N_x, Nt))

# Propagate the flow
for n in range(Nt-1):
    u[n+1, :, n+1] = u[n, :, n+1] + dt * (0.5 * (u[n, :, n+1] - u[n, :, n]) + (u[n, :, n+1] - u[n, :, n-1]) / dx + (v[n, :, n+1] - v[n, :, n]) / dy)
    v[n+1, :, n+1] = v[n, :, n+1] + dt * (0.5 * (v[n, :, n+1] - v[n, :, n]) + (u[n, :, n+1] - u[n, :, n]) / dy + (v[n, :, n+1] - v[n, :, n-1]) / dy)
    w[n+1, :, n+1] = w[n, :, n+1] + dt * (0.5 * (w[n, :, n+1] - w[n, :, n]) + (w[n, :, n+1] - w[n, :, n-1]) / dx + (u[n, :, n+1] - u[n, :, n]) / dy + (v[n, :, n+1] - v[n, :, n]) / dy)
    
    # Apply the perturbations
    u[n+1, :, n+1] += eps[n, :, n]
    v[n+1, :, n+1] += eps[n, :, n]
    w[n+1, :, n+1] += eps[n, :, n]

# Compute the Sobol sensitivity
S = pw.sobol_sensitivity(u, v, w, 'WS_eff', x, y)

# Save the sensitivity
np.save('Sobol_Sensitivity.npy', S)

# Plot the sensitivity
plt.imshow(S, cmap='coolwarm', origin='lower')
plt.colorbar()
plt.show()
```

This script first initializes the flow field and wave, then propagates the flow over time, applying the perturbations at each time step. After propagation, it computes the Sobol sensitivity of WS_eff as a function of x and y, and saves the sensitivity to a file. Finally, it plots the sensitivity using matplotlib.

Please note that you need to have pywake and numpy installed to run this script. Also, you can adjust the parameters (N_x, N_y, Lx, Ly, T, Nt, alpha, beta, sigma) to change the flow field and the sensitivity analysis.
