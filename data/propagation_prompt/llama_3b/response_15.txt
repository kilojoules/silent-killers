To generate a script that propagates realistic measurement uncertainties through pywake and saves a series of flow field images showing the sensitivity of the flow at different times to the inflow, we can use the following script. This script will use the `pywake` library to calculate the Sobol sensitivity of the wave speed (`WS_eff`) as a function of `x` and `y`, and then visualize the results.

```python
import numpy as np
import pywake as pw
import matplotlib.pyplot as plt
import matplotlib.animation as animation

# Define the inflow velocity profile
def inflow(x):
    return 1 + np.sin(x/10)

# Define the wave speed calculation
def WS_eff(x, y, inflow):
    return np.sqrt(inflow**2 + y**2)

# Generate a grid of x and y values
x = np.linspace(-10, 10, 100)
y = np.linspace(-10, 10, 100)
X, Y = np.meshgrid(x, y)

# Calculate the wave speed and its uncertainty
def calculate_WS_eff(X, Y, inflow, eps):
    WS_eff_grid = np.zeros((len(X), len(Y))
    WS_eff_uncertainty = np.zeros((len(X), len(Y))
    for i in range(len(X)):
        for j in range(len(Y)):
            WS_eff_grid[i, j] = WS_eff(X[i, j], Y[j], inflow(X[i, j]))
            WS_eff_uncertainty[i, j] = WS_eff_grid[i, j] * np.sqrt(eps**2 + Y[j]**2)
    return WS_eff_grid, WS_eff_uncertainty

# Propagate the uncertainty through the wave speed calculation
eps = 0.1  # Uncertainty in the inflow velocity profile
WS_eff_grid, WS_eff_uncertainty_grid = calculate_WS_eff(X, Y, inflow, eps)

# Save the flow field images
fig, axs = plt.subplots(10, figsize=(10, 10))
for i in range(10):
    axs[i].imshow(WS_eff_grid[i, :, :], cmap='RdBu', origin='lower')
    axs[i].set_title(f'Standard Deviation at t={i+1}')
    axs[i].set_xlabel('x')
    axs[i].set_ylabel('y')
plt.show()

# Calculate the Sobol sensitivity
sensitivity = pw.sobol(WS_eff_grid, eps=eps)
print(sensitivity)

# Save the Sobol sensitivity as a heatmap
sensitivity_grid = sensitivity[0]
plt.imshow(sensitivity_grid, cmap='RdBu', origin='lower')
plt.colorbar(label='Sobol Index')
plt.title('Sobol Sensitivity')
plt.xlabel('x')
plt.ylabel('y')
plt.show()
```

This script will generate a series of flow field images showing the standard deviation of the wave speed at different times (x and y values) and save the Sobol sensitivity as a heatmap.

Please note that the `pywake` library requires the `scipy` library to be installed. If it is not installed, you can install it using `pip install scipy`.

Also, the `inflow` function is a simple sine function, you can replace it with your own inflow velocity profile.

The `eps` variable represents the uncertainty in the inflow velocity profile, you can adjust it to change the level of uncertainty.
