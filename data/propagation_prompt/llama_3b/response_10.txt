I'll provide a script using Python and the PyWake library to propagate measurement uncertainties through the wake model and visualize the sensitivity of the flow field at different times.

```python
import numpy as np
import matplotlib.pyplot as plt
from pywake import wake_model, wakefield

# Define the wake model parameters
ws = wake_model.WakeModel()
ws.boundary_layer_momentum_coefficient = 0.002
ws.turbulence_intensity = 0.1
ws.density_ratio = 1.2
ws.length = 10
ws.width = 10
ws.depth = 1
ws.boundary_layer_momentum_coefficient = 0.002
ws.turbulence_intensity = 0.1
ws.density_ratio = 1.2

# Define the inflow flow field
def inflow(x, y):
    return np.sqrt(2 * 0.5 * 9.81 * 1)  # m/s

# Define the measurement uncertainty
def measurement_uncertainty(x, y):
    return 0.01 * inflow(x, y)  # m/s

# Propagate the uncertainty through the wake model
def propagate_uncertainty(x, y):
    wake = wakefield.WakeField(ws, inflow(x, y), measurement_uncertainty(x, y))
    return wake

# Create a grid of points to evaluate the wake
x = np.linspace(0, 10, 100)
y = np.linspace(0, 10, 100)
X, Y = np.meshgrid(x, y)

# Evaluate the wake field and uncertainty at each point
wake = propagate_uncertainty(X, Y)
uncertainty = wake.uncertainty

# Plot the wake field and uncertainty
plt.figure(figsize=(10, 10))
plt.contourf(X, Y, wake.omega, levels=20, cmap='viridis')
plt.colorbar(label='Wake Speed')
plt.title('Wake Field')
plt.show()

plt.figure(figsize=(10, 10))
plt.contourf(X, Y, uncertainty, levels=20, cmap='viridis')
plt.colorbar(label='Uncertainty')
plt.title('Uncertainty')
plt.show()

# Plot the sensitivity of the wake speed to the inflow at different times
for t in np.linspace(0, 10, 5):
    wake = wake_model.WakeModel()
    wake.boundary_layer_momentum_coefficient = 0.002
    wake.turbulence_intensity = 0.1
    wake.density_ratio = 1.2
    wake.length = 10
    wake.width = 10
    wake.depth = 1
    wake.t = t
    wake.calculate()
    wakefield = wakefield.WakeField(wake, inflow(X, Y), measurement_uncertainty(X, Y))
    uncertainty = wakefield.uncertainty
    plt.figure(figsize=(10, 10))
    plt.contourf(X, Y, wake.omega, levels=20, cmap='viridis')
    plt.colorbar(label='Wake Speed')
    plt.title(f'Wake Field at t={t}')
    plt.show()
    plt.figure(figsize=(10, 10))
    plt.contourf(X, Y, uncertainty, levels=20, cmap='viridis')
    plt.colorbar(label='Uncertainty')
    plt.title(f'Uncertainty at t={t}')
    plt.show()
```

This script first defines the wake model parameters and the inflow flow field. It then defines a function to propagate the measurement uncertainty through the wake model. The wake field and uncertainty are evaluated at a grid of points and plotted using contour plots. Finally, the sensitivity of the wake speed to the inflow at different times is plotted.

Please note that the `pywake` library uses the `wake_model` module, which is not a part of the standard library. You may need to install it separately using pip: `pip install pywake`. Also, the `wakefield` module is part of the `pywake` library, so you don't need to install it separately.

Please note that the wake model parameters and the inflow flow field are simplified for demonstration purposes. In a real-world scenario, you would need to use more complex models and take into account more factors such as the shape of the obstacle, the type of flow, and the surrounding environment.
