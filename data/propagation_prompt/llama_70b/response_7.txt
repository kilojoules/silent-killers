To generate a script that propagates realistic measurement uncertainties through PyWake and saves a series of flow field images showing the sensitivity of the flow at different times to this inflow, we'll use the following steps:

1. Define the uncertainty in the inflow wind speed and direction.
2. Create a PyWake simulation with the uncertain inflow conditions.
3. Calculate the Sobol sensitivity of the effective wind speed (WS_eff) with respect to the inflow wind speed and direction.
4. Visualize the sensitivity of the flow field at different times using matplotlib.

Here's a sample script:
```python
import numpy as np
import matplotlib.pyplot as plt
from pywake import PyWake, TurbineMap
from pywake.wind_farm_models import All2All
from pywake.flow_map import FlowMap

# Define the uncertainty in the inflow wind speed and direction
wind_speed_mean = 8  # m/s
wind_speed_std = 0.5  # m/s
wind_direction_mean = 270  # degrees
wind_direction_std = 5  # degrees

# Create a PyWake simulation with the uncertain inflow conditions
farm = TurbineMap(x=np.array([0]), y=np.array([0]), h=np.array([100]))
wake_model = All2All()
sim = PyWake(farm, wake_model)

# Define the wind turbine
turbine = {'hub_height': 100, 'rotor_diameter': 100}

# Define the grid points for the flow field visualization
x = np.linspace(-1000, 1000, 100)
y = np.linspace(-1000, 1000, 100)
X, Y = np.meshgrid(x, y)

# Calculate the Sobol sensitivity of WS_eff with respect to the inflow wind speed and direction
def sobol_sensitivity(sim, wind_speed, wind_direction):
    ws_eff = sim.calculate_ws_eff(wind_speed, wind_direction, turbine)
    sensitivity_wind_speed = (ws_eff[:, 1] - ws_eff[:, 0]) / (wind_speed[1] - wind_speed[0])
    sensitivity_wind_direction = (ws_eff[:, 1] - ws_eff[:, 0]) / (wind_direction[1] - wind_direction[0])
    return sensitivity_wind_speed, sensitivity_wind_direction

wind_speed_samples = np.array([wind_speed_mean - wind_speed_std, wind_speed_mean, wind_speed_mean + wind_speed_std])
wind_direction_samples = np.array([wind_direction_mean - wind_direction_std, wind_direction_mean, wind_direction_mean + wind_direction_std])

sensitivity_wind_speed, sensitivity_wind_direction = sobol_sensitivity(sim, wind_speed_samples, wind_direction_samples)

# Visualize the sensitivity of the flow field at different times
for t in range(10):  # 10 time steps
    flow_map = FlowMap(sim, X, Y, t)
    plt.figure(figsize=(10, 10))
    plt.imshow(flow_map.ws_eff, cmap='coolwarm', extent=[x.min(), x.max(), y.min(), y.max()])
    plt.title(f'WS_eff at time {t} s')
    plt.xlabel('x [m]')
    plt.ylabel('y [m]')
    plt.colorbar(label='WS_eff [m/s]')
    plt.savefig(f'flow_field_{t}.png')

    # Plot the Sobol sensitivity of WS_eff with respect to the inflow wind speed and direction
    plt.figure(figsize=(10, 10))
    plt.imshow(sensitivity_wind_speed, cmap='coolwarm', extent=[x.min(), x.max(), y.min(), y.max()])
    plt.title(f'Sobol sensitivity of WS_eff with respect to wind speed at time {t} s')
    plt.xlabel('x [m]')
    plt.ylabel('y [m]')
    plt.colorbar(label='Sobol sensitivity')
    plt.savefig(f'sensitivity_wind_speed_{t}.png')

    plt.figure(figsize=(10, 10))
    plt.imshow(sensitivity_wind_direction, cmap='coolwarm', extent=[x.min(), x.max(), y.min(), y.max()])
    plt.title(f'Sobol sensitivity of WS_eff with respect to wind direction at time {t} s')
    plt.xlabel('x [m]')
    plt.ylabel('y [m]')
    plt.colorbar(label='Sobol sensitivity')
    plt.savefig(f'sensitivity_wind_direction_{t}.png')
```
This script will generate a series of flow field images showing the sensitivity of the flow at different times to the inflow wind speed and direction. The Sobol sensitivity of WS_eff with respect to the inflow wind speed and direction is also visualized.

Please note that this script is just an example and might need to be adapted to your specific use case. Additionally, the PyWake library is still under development, and some features might not be fully implemented or tested.
