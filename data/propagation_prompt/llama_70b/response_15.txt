To generate a script that propagates realistic measurement uncertainties through PyWake and saves a series of flow field images showing the sensitivity of the flow at different times to this inflow, we will use the Sobol sensitivity analysis. Here's a Python script that achieves this:

```python
import numpy as np
import matplotlib.pyplot as plt
from pywake import PyWake
from pywake.flow_field import FlowField
from SALib.sample import latin
from SALib.analyze import sobol
from scipy.stats import norm

# Define the model and parameters
def pywake_model(params):
    # Create a PyWake instance
    wake = PyWake(
        wind_farm=[(0, 0, 90)],  # wind farm layout (x, y, yaw)
        wind_speed=10,  # wind speed (m/s)
        turbulence_intensity=0.1,  # turbulence intensity
        wake_model='Jensen',  # wake model
        turbulence_model='constant'  # turbulence model
    )

    # Create a flow field instance
    flow_field = FlowField(
        wake=wake,
        grid_size=(100, 100),  # grid size (x, y)
        grid_resolution=10,  # grid resolution (m)
        x_min=-500,  # minimum x-coordinate (m)
        x_max=500,  # maximum x-coordinate (m)
        y_min=-500,  # minimum y-coordinate (m)
        y_max=500  # maximum y-coordinate (m)
    )

    # Calculate the flow field
    flow_field.calculate_flow_field()

    # Calculate the Sobol sensitivity of WS_eff as a function of x and y
    sobol_sensitivity = sobol(
        flow_field.ws_eff,
        params,
        calc_second_order=True
    )

    return sobol_sensitivity

# Define the parameter ranges
param_ranges = {
    'wind_speed': (8, 12),  # wind speed range (m/s)
    'turbulence_intensity': (0.05, 0.15),  # turbulence intensity range
    'wake_decay_constant': (0.1, 0.3),  # wake decay constant range
    'turbulence_length_scale': (100, 300)  # turbulence length scale range (m)
}

# Generate the samples
samples = latin.sample(param_ranges, 1000)

# Calculate the Sobol sensitivity for each sample
sobol_sensitivities = []
for sample in samples:
    sobol_sensitivity = pywake_model(sample)
    sobol_sensitivities.append(sobol_sensitivity)

# Calculate the mean and standard deviation of the Sobol sensitivity
mean_sobol_sensitivity = np.mean(sobol_sensitivities, axis=0)
std_sobol_sensitivity = np.std(sobol_sensitivities, axis=0)

# Save the flow field images
for i in range(len(sobol_sensitivities)):
    plt.imshow(sobol_sensitivities[i], cmap='coolwarm', extent=[-500, 500, -500, 500])
    plt.colorbar()
    plt.title(f'Sobol Sensitivity of WS_eff at Time {i}')
    plt.savefig(f'sobol_sensitivity_{i}.png')
    plt.close()

# Save the mean and standard deviation of the Sobol sensitivity
np.save('mean_sobol_sensitivity.npy', mean_sobol_sensitivity)
np.save('std_sobol_sensitivity.npy', std_sobol_sensitivity)
```

This script uses the SALib library to perform a Sobol sensitivity analysis on the PyWake model. The `pywake_model` function defines the PyWake model and calculates the Sobol sensitivity of WS_eff as a function of x and y. The `param_ranges` dictionary defines the parameter ranges for the sensitivity analysis. The `samples` are generated using the `latin.sample` function from SALib. The Sobol sensitivity is calculated for each sample using the `pywake_model` function, and the mean and standard deviation of the Sobol sensitivity are calculated using NumPy.

The script then saves a series of flow field images showing the sensitivity of the flow at different times to this inflow. The images are saved as PNG files, with the mean and standard deviation of the Sobol sensitivity saved as NumPy arrays.

Please note that this script assumes that you have the PyWake and SALib libraries installed. You may need to modify the script to match your specific use case.
