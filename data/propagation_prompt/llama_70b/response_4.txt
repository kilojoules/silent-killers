I'll provide a script that propagates realistic measurement uncertainties through PyWake and saves a series of flow field images showing the sensitivity of the flow at different times to the inflow. I'll use the Sobol sensitivity analysis to compute the sensitivity of WS_eff as a function of x and y.

Please note that I'll assume you have PyWake installed and configured properly. If not, please refer to the PyWake documentation for installation and setup instructions.

**Script:**
```python
import numpy as np
import matplotlib.pyplot as plt
from pywake import PyWake, FlowField
from pywake.utils import SobolSensitivity
from scipy.stats import norm

# Define the wind farm layout
x = np.array([0, 100, 200, 300, 400, 500])  # x-coordinates of turbines
y = np.array([0, 0, 0, 0, 0, 0])  # y-coordinates of turbines

# Define the inflow conditions
u_inflow = 10.0  # inflow wind speed (m/s)
ti_inflow = 0.1  # inflow turbulence intensity (%)

# Define the uncertainty bounds for the inflow conditions
u_inflow_uncertainty = 0.5  # uncertainty in inflow wind speed (m/s)
ti_inflow_uncertainty = 0.02  # uncertainty in inflow turbulence intensity (%)

# Create a PyWake instance
wake = PyWake(x, y, u_inflow, ti_inflow)

# Define the time steps for the flow field analysis
time_steps = np.arange(0, 3600, 60)  # 1-minute time steps for 1 hour

# Create a SobolSensitivity instance
sobol = SobolSensitivity(wake, time_steps)

# Define the parameters for the Sobol sensitivity analysis
parameters = {
    'u_inflow': {'distribution': 'normal','mean': u_inflow,'stddev': u_inflow_uncertainty},
    'ti_inflow': {'distribution': 'normal','mean': ti_inflow,'stddev': ti_inflow_uncertainty}
}

# Run the Sobol sensitivity analysis
sobol_results = sobol.run(parameters)

# Create a figure to store the flow field images
fig, axes = plt.subplots(len(time_steps), 2, figsize=(12, 6*len(time_steps)))

# Loop through the time steps and create flow field images
for i, t in enumerate(time_steps):
    # Compute the flow field at the current time step
    flow_field = wake.compute_flow_field(t)
    
    # Compute the Sobol sensitivity of WS_eff at the current time step
    sobol_sensitivity = sobol_results[t]['WS_eff']
    
    # Create a contour plot of the flow field
    axes[i, 0].contourf(flow_field.x, flow_field.y, flow_field.u, levels=20)
    axes[i, 0].set_title(f'Flow Field at t={t} s')
    
    # Create a contour plot of the Sobol sensitivity
    axes[i, 1].contourf(flow_field.x, flow_field.y, sobol_sensitivity, levels=20)
    axes[i, 1].set_title(f'Sobol Sensitivity of WS_eff at t={t} s')

# Save the figure to a file
plt.savefig('sobol_sensitivity.png')

# Close the figure
plt.close(fig)
```
This script will generate a series of flow field images showing the sensitivity of the flow at different times to the inflow, as well as the Sobol sensitivity of WS_eff as a function of x and y. The images will be saved to a file named `sobol_sensitivity.png`.

Please note that this script assumes that the PyWake instance is created with the default settings. You may need to modify the script to match your specific use case.

Also, the Sobol sensitivity analysis can be computationally intensive, so you may need to adjust the number of time steps or the number of samples used in the analysis to achieve a reasonable computation time.
