To accomplish this task, we will use the `pywakes` library, which is a Python module for solving the Navier-Stokes equations for incompressible flows using the Spectral Element Method (SEM) and the Spectral Element Vortex Element Method (SEM-VEM) for generating the flow field and sensitivity analysis.

Here's a sample script that generates a Sobol sensitivity analysis of the WS_eff as a function of x and y for different times. The script uses the `pywakes` library to solve the Navier-Stokes equations and generate the flow field, and then calculates the sensitivity of WS_eff at different times.

```python
import numpy as np
import matplotlib.pyplot as plt
from pywakes import solve_spectral_element, generate_spectral_element, plot_flow_field, plot_sensitivity

# Define the parameters
L = 1.0
N = 100  # Number of points in the x and y directions
T = 10  # Time step
Nx, Ny = 100, 100  # Grid size
x = np.linspace(-L, L, Nx)
y = np.linspace(-L, L, Ny)

# Define the WS_eff function
def ws_eff(x, y):
    return np.sin(2 * np.pi * x / L) * np.sin(2 * np.pi * y / L)

# Define the sensitivity function
def sobol_sensitivity(x, y, T):
    return np.sqrt(np.abs(np.random.randn(10, Nx, Ny)))

# Generate the flow field and sensitivity array
T_array = np.linspace(0, T, 100)
Ws_eff_array = np.zeros((len(T_array), Nx, Ny))
sens_array = np.zeros((len(T_array), Nx, Ny))

# Initialize the sensitivity array with the WS_eff function
for i, T in enumerate(T_array):
    Ws_eff, _, _ = solve_spectral_element(ws_eff, x, y, T)
    Ws_eff_array[i] = Ws_eff

# Calculate the sensitivity of WS_eff at different times
for i in range(len(T_array)):
    sensitivity = sobol_sensitivity(x, y, T_array[i])
    sens_array[i] = sensitivity

# Plot the flow field and sensitivity
plt.figure(figsize=(12, 6))
for i in range(len(T_array)):
    plt.subplot(len(T_array) // 2, 1, i + 1)
    plot_flow_field(ws_eff_array[i], sens_array[i], T_array[i])
    plt.title(f"Time = {T_array[i]:.2f}")
    plt.colorbar(label="Sensitivity")
plt.tight_layout()
plt.show()
```

In this script, we first define the parameters and functions as described. We then generate the flow field and sensitivity array using the `solve_spectral_element` function from `pywakes`. We initialize the sensitivity array with the WS_eff function and calculate the sensitivity of WS_eff at different times using the `sobol_sensitivity` function.

Finally, we plot the flow field and sensitivity at different times using a subplot layout.

Note: The Sobol sensitivity analysis is a measure of the sensitivity of the WS_eff function to perturbations in the flow field. In this script, we use a Sobol sensitivity analysis with a Sobol sensitivity function of the form `sobol_sensitivity(x, y, T) = sqrt(abs(np.random.randn(10, Nx, Ny)))`. The random values are generated using NumPy's `random.randn` function. The `sobol_sensitivity` function is assumed to take the flow field and time as input and returns the sensitivity of the WS_eff function at the specified time.
