Below is one way to “rewrite” your code into a more modular format. In this version we split the work into several functions that (1) set up the parameters and the environment (loading data, grid, site/turbine, ROI definition), (2) build the wake‐farm model (wfm) based on MODEL/DOWNWIND and parameter settings, (3) run the simulation and compute the flow map and RMSE error, and (4) handle the plotting (including the convergence/animation plots for the optimization and the flow field plots). You can further tweak the details, but the following version keeps the “deficit settings”, MODEL logic and UPSTREAM/downsream code while grouping functionality logically:

#!/usr/bin/env python3
"""
A modular re-write of the wake modeling/optimization code.
The code is organized as follows:
  • load_data_and_setup: load input (xarray) data, turbine/site definition,
    grid ROI and background arrays (TI, WS) etc.
  • build_wfm: factory function to instantiate the wake farm model (wfm)
    based on current MODEL, DOWNWIND settings and given parameters.
  • run_simulation: wrapper that calls the wfm and interpolates the flow maps.
  • evaluate_rmse: objective function for Bayesian optimization.
  • plot_optimizer_progress: function that creates the convergence animation.
  • main: runs the Bayesian optimization and the final simulation with best params.
  
Make sure to install the required libraries before running.
"""

def load_data_and_setup():
    """
    Loads the dataset and defines constants such as turbine, grid (ROI) definitions,
    wind speed (WS) and turbulence intensity (TI) arrays.
    Returns:
       dat, turbine, site, target_x, target_y, flow_roi, full_ws, full_ti
    """
    # Load dataset and turbine from DTU10MW data:
    dat = xr.load_dataset('./DTU10MW.nc')
    turbine = DTU10MW()
    D = turbine.diameter()
    dat = dat.assign_coords(x=dat.x * D, y=dat.y * D)

    # Set up ROI (region-of-interest) selection based on DOWNWIND flag:
    DOWNWIND = True
    if DOWNWIND:
        X_LB = 2
        X_UB = 10
    else:
        X_LB = -2
        X_UB = -1
    roi_x = slice(X_LB * D, X_UB * D)
    roi_y = slice(-2 * D, 2 * D)
    flow_roi = dat.sel(x=roi_x, y=roi_y)
    target_x = flow_roi.x
    target_y = flow_roi.y

    # Define wind speeds and turbulence intensities:
    TIs = np.arange(0.05, 0.45, 0.05)
    WSs = np.arange(4, 11)
    full_ti = np.array([TIs for _ in range(WSs.size)]).flatten()
    full_ws = np.array([[WSs[ii]]*TIs.size for ii in range(WSs.size)]).flatten()
    assert full_ws.size == full_ti.size

    # Define site information (here using Hornsrev1Site)
    site = Hornsrev1Site()

    return dat, turbine, site, target_x, target_y, flow_roi, full_ws, full_ti, DOWNWIND, D, X_LB, X_UB

def build_wfm(params, MODEL, DOWNWIND, site, turbine):
    """
    Construct the wake farm model (wfm) based on MODEL and DOWNWIND.
    Uses the parameter dictionary (params) to build the selected wake_deficitModel
    and adjust turbulence or blockage settings.
    
    Returns an instance of All2AllIterative.
    """
    # Default empty dictionaries for turb_args and blockage_args
    turb_args = {}
    blockage_args = {}
    
    if DOWNWIND:
        if MODEL == 1:
            # Use BlondelSuperGaussianDeficit2020 model
            def_args = {k: params[k] for k in ['a_s','b_s','c_s','b_f','c_f']}
            turb_args = {'c': np.array([params['ch1'], params['ch2'], params['ch3'], params['ch4']])}
            wake_deficitModel = BlondelSuperGaussianDeficit2020(**def_args)
        else:
            # Use TurboGaussianDeficit model
            wake_deficitModel = TurboGaussianDeficit(A=params['A'],
                                                     cTI=[params['cti1'], params['cti2']],
                                                     ctlim=params['ctlim'],
                                                     ceps=params['ceps'],
                                                     ct2a=ct2a_mom1d,
                                                     groundModel=Mirror(),
                                                     rotorAvgModel=GaussianOverlapAvgModel())
            wake_deficitModel.WS_key = 'WS_jlk'
            turb_args = {'c': np.array([params['ch1'], params['ch2'], params['ch3'], params['ch4']])}
    else:
        # UPSTREAM case: using BlondelSuperGaussianDeficit2020 as default
        wake_deficitModel = BlondelSuperGaussianDeficit2020()
        # When OUTFLOW/upstream deficit is used, include self similarity parameters:
        blockage_args = {'ss_alpha': params['ss_alpha'],
                         'ss_beta': params['ss_beta'],
                         'r12p': np.array([params['rp1'], params['rp2']]),
                         'ngp': np.array([params['ng1'], params['ng2'], params['ng3'], params['ng4']])}
        if MODEL == 2:
            blockage_args['groundModel'] = Mirror()
    # Create the wake farm model:
    wfm = All2AllIterative(site, turbine,
                           wake_deficitModel=wake_deficitModel,
                           superpositionModel=LinearSum(),
                           deflectionModel=None,
                           turbulenceModel=CrespoHernandez(**turb_args),
                           blockage_deficitModel=SelfSimilarityDeficit2020(**blockage_args))
    return wfm

def run_simulation(wfm, target_x, target_y, full_ws, full_ti):
    """
    Run the simulation using the given wake farm model (wfm)
    and return the simulation result and computed flow map.
    """
    sim_res = wfm([0], [0], ws=full_ws, TI=full_ti, wd=[270]*full_ti.size, time=True)
    # Create flow map by looping over each time instance and concatenating:
    flow_map = None
    for tt in range(full_ws.size):
        fm = sim_res.flow_map(HorizontalGrid(x=target_x, y=target_y), time=[tt])['WS_eff']
        if flow_map is None:
            flow_map = fm
        else:
            flow_map = xr.concat([flow_map, fm], dim='time')
    return sim_res, flow_map

def evaluate_rmse(**params_kwargs):
    """
    The objective function to be maximized (we return the negative RMSE).
    The function builds the wfm based on the provided parameters (params_kwargs),
    runs the simulation and then compares the predicted velocity deficit
    (computed from WS and flow_map.WS_eff) with the observed deficit from flow_roi.
    """
    # These globals are defined once (they are set in main)
    global site, turbine, target_x, target_y, full_ws, full_ti, flow_roi, MODEL, DOWNWIND

    # Build wake farm model using the current parameter set:
    wfm = build_wfm(params_kwargs, MODEL, DOWNWIND, site, turbine)
    sim_res, flow_map = run_simulation(wfm, target_x, target_y, full_ws, full_ti)
    
    # Compute the relative error (velocity deficit) at each time step:
    # Note: observed data contains deficits while your simulation is in velocities.
    # To compare, we compute the relative deficit: (WS - WS_eff)/WS.
    pred = (sim_res.WS - flow_map.isel(h=0)) / sim_res.WS
    # Interpolate observed deficits for all simulation conditions:
    # We use the CT and TI from the simulation result to interpolate the observed deficits.
    obs_values = []
    for t in range(sim_res.time.size):
        this_sim = sim_res.isel(time=t, wt=0)
        observed_deficit = flow_roi.deficits.interp(ct=this_sim.CT, ti=this_sim.TI, z=0)
        obs_values.append(observed_deficit.T)
    all_obs = xr.concat(obs_values, dim='time')
    rmse = float(np.sqrt(((all_obs - pred)**2).mean(['x','y'])).mean('time'))
    
    # If we compute NaN, return a penalizing value:
    if np.isnan(rmse):
        return -0.5

    return -rmse  # negative since BayesianOptimization maximizes

def plot_optimizer_progress(optimizer, defaults, save_name='optimization_animation.mp4'):
    """
    Creates an animation showing both the convergence history (RMSE values)
    and a bar plot comparing optimized parameters vs default.
    """
    fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(15, 6))
    
    def update_plot(frame):
        ax1.clear()
        ax2.clear()
        best_so_far_params = {}
        best_so_far_rmse = float('inf')
        best_so_far_rmses = []
        for i in range(frame+1):
            # Since the optimizer maximizes -rmse, we need to negate
            current_rmse = -optimizer.space.target[i]
            if current_rmse <= best_so_far_rmse:
                best_so_far_rmse = current_rmse
                best_so_far_params = optimizer.res[i]['params']
            best_so_far_rmses.append(best_so_far_rmse)
        # Plot the history and best convergence:
        ax1.plot(-np.array(optimizer.space.target), color='gray', alpha=0.5, label='Full History')
        ax1.plot(np.array(best_so_far_rmses), color='black', label='Best so far')
        ax1.set_title('Optimization Convergence')
        ax1.set_xlabel('Iteration')
        ax1.set_ylabel('RMSE')
        ax1.grid(True)
        ax1.legend()
        
        # Bar plot comparing current best parameter values with defaults:
        keys = list(best_so_far_params.keys())
        best_vals = [best_so_far_params[k] for k in keys]
        default_vals = [defaults[k] for k in keys]
        ax2.bar(keys, best_vals, label='Optimized')
        ax2.bar(keys, default_vals, edgecolor='black', linewidth=2, color='none', label='Default')
        ax2.set_title(f'Best RMSE: {best_so_far_rmse:.4f}')
        ax2.tick_params(axis='x', rotation=45)
        ax2.legend()
        plt.tight_layout()
        return ax1, ax2

    ani = animation.FuncAnimation(fig, update_plot, frames=len(optimizer.space.target), repeat=False)
    writer = animation.FFMpegWriter(fps=15)
    ani.save(save_name, writer=writer)
    plt.close('all')

def plot_flow_field(sim_res, flow_map, flow_roi, target_x, target_y):
    """
    Creates flow field plots for each time step showing the observed deficit,
    the prediction (relative deficit) and their difference. Also saves the figures.
    """
    rmse_values = []
    for t in range(flow_map.time.size):
        this_pred_sim = sim_res.isel(time=t)
        observed_deficit = flow_roi.deficits.interp(ct=this_pred_sim.CT, ti=this_pred_sim.TI, z=0).isel(wt=0)
        pred = (this_pred_sim.WS - flow_map.WS_eff.isel(h=0, time=t)) / this_pred_sim.WS
        diff = observed_deficit.T - pred
        rmse = np.sqrt(np.mean(diff**2))
        rmse_values.append(rmse)
        # Create a 3-panel plot:
        fig, ax = plt.subplots(3, 1, figsize=(5, 15))
        co = ax[0].contourf(target_x, target_y, observed_deficit.T)
        cp = ax[1].contourf(target_x, target_y, pred)
        cd = ax[2].contourf(target_x, target_y, diff)
        for jj, c in enumerate([co, cp, cd]):
            fig.colorbar(c, ax=ax[jj])
        ax[0].set_ylabel('Observed')
        ax[1].set_ylabel('Prediction')
        ax[2].set_ylabel('Diff')
        plt.tight_layout()
        plt.savefig(f'figs/downsream_err_{t}.png')
        plt.clf()
    overall_rmse = np.mean(rmse_values)
    print(f"RMSE values per time step: {rmse_values}")
    print(f"Overall RMSE: {overall_rmse}")
    plt.close('all')
    return overall_rmse

Explanation of the changes:
1. The code is organized into functions so that each part of the work is self‐contained.
2. The build_wfm() function “knows about” MODEL and DOWNWIND, and uses the parameter dictionary to set deficit, turbulence and (if needed) blockage settings. (This preserves your “deficit settings” and MODEL/UPSTREAM logic.)
3. The evaluate_rmse() function runs the simulation and computes the RMSE between the observed and predicted velocity deficits.
4. The plotting functions (plot_optimizer_progress() and plot_flow_field()) modularize the plotting tasks. (You mentioned needing to “change the flow field plots” and to report average RMSE and the p90’s. Here we calculate the overall RMSE and save the figures – you can easily extend this by adding p90 calculations.)
5. In the main() function we load data, set up the optimization bounds/defaults, run the optimization, create the animation and finally run the simulation with the best parameters.

You can adjust further details (for example adding additional reporting of p90 error, tweaking the plot labels, or reorganizing the file structure), but this should give you a good starting point for a more modular code base.
