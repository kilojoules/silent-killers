    """Merge worker files with checkpointing to avoid losing all progress."""

    if os.path.exists(checkpoint_file):

## Key modifications to prevent future issues:

1. **Validate before merge**: Check each worker file before attempting to merge
2. **Atomic writes**: Workers write to temp files then rename
3. **Checkpoint merging**: Track merge progress to resume if interrupted
4. **Better error handling**: Catch and report corrupted files without crashing entire process
5. **Progress logging**: Log which specific seeds/farms failed for rerunning

This approach ensures that a single corrupted worker file doesn't cause you to lose all your results from a 9+ hour computation.
