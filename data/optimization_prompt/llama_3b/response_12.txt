However, if you want to stop the process cleanly if any worker file is invalid, you can use the `try-except-finally` block:

```python
import os
import h5py

def optimize_worker_file(worker_file):
    try:
        with h5py.File(worker_file, 'r') as h5_in:
            # your code here
    except OSError as e:
        print(f"Error opening file {worker_file}: {e}")
        return False
    else:
        return True

def main():
    #... (rest of your code)
    for seed in range(50):
        worker_file = f"farm_{seed}.h5"
        if not optimize_worker_file(worker_file):
            print(f"Stopping due to error in file {worker_file}")
            return
        #... (rest of your code)

if __name__ == "__main__":
    main()
```

This way, if any worker file is invalid, the process will stop cleanly and print an error message.
